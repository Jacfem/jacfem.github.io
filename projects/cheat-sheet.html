    Arrays

    Adding Items

  • .push : Add a new item to the end of an array. You can also use <<
  • .unshift :Add a new item to beginning of an array.
  • .insert :
  • Add a new element to an array at any position. array.insert(2, ‘Jackie’) will insert ‘Jackie’ in the 3rd position. You can add multiple items this way.

    • Removing Items

  • .pop :Removes the last element from the array and returns it, or nil if the array is empty. You can specify a number such as array.pop(2) which will return an array of the last 2 elements.
  • .shift :Retrieves and removes the first element from the array.
  • .delete_at(index) :Deletes an element at a particular index.
  • .delete(element) : Deletes a particular element anywhere in the array.
  • .slice(index) : Deletes the element at the given index. You can specify a range.
  • .compact :Removes nil values from an array.
  • .uniq :Removes duplicate elements from an array.

    • Iterators

  • .each :Leaves array unchanged and returns the values of the code block.
  • .each_index :Same as each, but passes the index of the element instead of the element itself.
  • .reverse_each :Iterates over the elements in an array in reverse order.
  • .map {|item| block of code} :Outputs an array with values returned by the block.
  • .map! {|item| block of code} :Outputs an array with the values replaced by the value returned by each block.

    • Selecting Items

  • .select{|a| code block for condition} : Selects elements from an array according to the criteria in the code block. Will return a new array unless you use select!, the destructive form, in which case it modifies the original array.
  • .reject{|a| code block for condition} : Same as with select, except it will reject the values based on the code block you specify. Again, you can use a destructive form with reject!

    • Array/Element Reference

  • [index] :Returns the element at the array’s index specified.
  • [range] :Returns a subarray containing the elements from that range of indexes, inclusive.
  • .count :Returns number of elements in the array. You can include a code block that will return the number of elements for which the code block returns true.

    • Other useful methods for arrays!

  • .concat : Adds elements to the array.
  • .cycle(number) {code block} : Calls the code block on the array a certain number of times.
  • .join(separator) : Returns a string with each element from the array separated by the given separator.
  • .reverse : Returns a new array containing the array’s elements in reversed order.
  • .flatten :Returns a new array that is a one dimensional flattening of the initial array. If you’d like, you can specify a level to flatten, such as array.flatten(1) to flatten one level. Otherwise, it will flatten all levels.
  • .sample(optionalNumber) : Chooses a random element or random number of elements from the array.
  • .sort : Returns a new sorted array. Must use destructive version sort! if you want to modify the original array.
